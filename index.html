FAQ

❓ How do I start drawing?

Use CTRL+ALT+Z to toggle the drawing window. Then use hotkeys inside it.

🎨 How do I change brush size?

Press O to increase, I to decrease brush size.

🌈 Can I pick a color?

Yes! Use CTRL+SHIFT+C to pick a color directly from the screen.

💾 How big is the app?

The installer is ~50MB because it includes all required libraries to run smoothly on any PC. So don't go and cry about it on Reddit😉
